🎯 ¿Qué es Hatsik?

Hatsik (del maya: "dividir, compartir") es una app web para organizar eventos colectivos — asados, posadas, cumpleaños, picnics. Resuelve el caos de "¿qué llevo?" que normalmente se coordina en hilos de WhatsApp.

Un organizador crea un evento, arma la lista de cosas que se necesitan, y comparte un link o QR. Los invitados piden acceso, se les aprueba, y luego pueden reclamar qué van a llevar. La lista se actualiza para que nadie repita ni se olvide nada.

📝 Resumen ejecutivo

Hatsik se desarrolló del 24 de mayo al 13 de julio de 2026 (51 días). Se construyó como un monolito Django con templates renderizados en servidor e interactividad vía HTMX — sin SPA, sin JavaScript complejo, sin API REST separada.

El proyecto siguió un roadmap de 6 fases secuenciales, cada una con su propio branch, PR, y suite de tests. Cada fase se diseñó primero en Stitch (herramienta de diseño con AI de Google) y luego se implementó replicando fielmente el HTML generado.

El resultado es una app funcional y testeada que resuelve el problema de coordinación de listas compartidas para eventos sociales, lista para ser desplegada a producción.
