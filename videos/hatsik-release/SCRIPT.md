## Beat notes

1. **Intro** — brand title card on cream, logo + hero headline, red accent on "¿qué llevo?". No screen recording.
2. **Crear evento** — clip: dashboard → clic "Crear evento" → formulario (nombre "Asado de fin de año", fecha, descripción) tecleado rápido → clic "Crear Evento" → aterriza en el detalle del evento.
3. **Añadir ítems** — clip: modal "Añadir ítem" (nombre, cantidad, unidad) → la lista se llena con 3-4 ítems (time-lapse).
4. **Compartir** — clip: página /share/ → zoom al link público → clic "Copiar link" → feedback "¡Copiado!" → zoom al QR.
5. **Tomar un ítem** — clip: fila de ítem → clic "Asignarme" → confirmación inline ("¿Vas a traer…?") → "Yo lo llevo" / "Confirmar" → avatar aparece en la fila.
6. **Comprar** — clip: clic en el ícono de carrito → badge verde "Comprado" + barra de progreso del evento sube.
7. **Cierre** — brand card: "3. ¡Y listo!" + URL + CTA "Crea tu primer evento".
